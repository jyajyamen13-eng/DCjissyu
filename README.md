# 障害者雇用実習計画書システム

視覚的に見やすく、PDF出力も可能な実習計画書作成システムです。

## 🎯 主な機能

### ✨ 基本機能
- **Wordファイル読み込み**: 既存の.docxファイルをアップロードして自動入力
- **イニシャル変換**: 利用者名を自動的にイニシャルに変換（プライバシー保護）
- **美しいデザイン**: グラデーション背景と見やすいレイアウト
- **PDF出力**: ブラウザの印刷機能を使用してPDF保存可能
- **データ保存**: ローカルストレージに自動保存
- **データ読込**: 保存したデータの復元
- **レスポンシブデザイン**: PC・タブレット・スマホ対応

### 🔒 プライバシー保護機能
- **自動イニシャル変換**: 
  - 日本語名: 「山田太郎」→ 「山.太.」
  - 英語名: "John Smith" → "J.S."
- **ローカル保存**: データは端末内のみに保存
- **外部送信なし**: すべての処理がブラウザ内で完結

### 📋 自動計算機能
- 生年月日から年齢を自動計算
- 通所開始日から在籍月数を自動計算

## 🚀 使い方

### Wordファイルからの読み込み

1. **アップロードページにアクセス**
   - `upload.html` をブラウザで開く
   - または `index.html` から「Word読込」ボタンをクリック

2. **ファイルをアップロード**
   - ドラッグ＆ドロップで.docxファイルをアップロード
   - またはクリックしてファイルを選択

3. **自動変換**
   - 利用者名は自動的にイニシャルに変換されます
     - 例: 「山田太郎」→ 「山.太.」
     - 例: "John Smith" → "J.S."
   - その他の情報はそのままフォームに反映されます

4. **編集と保存**
   - フォームページに自動遷移
   - 必要に応じて編集
   - PDF出力またはデータ保存

### 1. ローカルで実行する場合

すべてのファイルを同じフォルダに配置し、`index.html` をブラウザで開いてください。

```bash
# ファイル構成
internship-form-system/
├── index.html          # メインHTML（フォーム構造）
├── upload.html         # Wordファイルアップロードページ
├── styles.css          # スタイルシート（デザイン）
├── script.js           # JavaScript（機能実装）
├── README.md           # このファイル
└── SETUP_GUIDE.md      # GitHubセットアップガイド
```

## 📖 操作方法

### PDF出力
1. フォームに必要事項を入力
2. 「📄 PDF出力」ボタンをクリック
3. 印刷ダイアログで「PDFとして保存」を選択

### データ保存
1. フォームに入力後、「💾 データ保存」ボタンをクリック
2. データはブラウザのローカルストレージに保存されます

### データ読込
1. 「📂 データ読込」ボタンをクリック
2. 前回保存したデータが復元されます

### フォームクリア
1. 「🗑️ クリア」ボタンをクリック
2. 確認ダイアログで「OK」を選択
3. すべての入力内容がクリアされます

## 🔒 セキュリティとプライバシー

- **ローカルストレージ**: データは端末内のみに保存
- **外部送信なし**: インターネット接続不要（初回読み込み後）
- **個人情報保護**: 利用者名は自動的にイニシャル化
- **オープンソース**: すべてのコードが確認可能

## 🌐 GitHub Pagesで公開する場合

詳細は `SETUP_GUIDE.md` を参照してください。

1. GitHubにリポジトリを作成
2. ファイルをアップロード
3. Settings → Pages でGitHub Pagesを有効化
4. 公開URLにアクセス

## 📦 技術スタック

- **HTML5**: セマンティックなマークアップ

## 🛠️ カスタマイズ

### 色の変更
`styles.css` の以下の部分を編集:

```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### 事業所情報の変更
`index.html` の連絡先セクションを編集:

```html
<input type="text" id="officeName" name="officeName" value="あなたの事業所名">
```
